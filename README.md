
# 🛡️ Ageis AI — Intelligent Crisis Response System

## 📌 Overview
**Ageis AI** is an advanced, AI-driven crisis response system designed for smart buildings, with a strong focus on the hospitality sector. The system enables **real-time emergency detection, predictive risk analysis, and dynamic evacuation planning** to ensure faster, safer, and more coordinated responses during critical situations.

It combines **artificial intelligence, real-time data processing, and 3D visualization** to assist both guests and staff in managing emergencies such as fire outbreaks, structural hazards, and other critical incidents.

---

## 🎯 Problem Statement
In large buildings like hotels, emergency response is often:
- Delayed due to lack of real-time awareness  
- Inefficient due to static evacuation plans  
- Poorly coordinated between staff and management  
- Risky due to unpredictable hazard spread  

**Ageis AI solves this by providing an intelligent, adaptive, and real-time crisis management system.**

---

## 🚀 Key Features

### 🔍 Real-Time Emergency Detection
- Detects fire, smoke, and anomalies using sensor and CCTV inputs  
- Uses AI-based computer vision for accurate threat identification  
- Automatically triggers alerts and response mechanisms  

### 🧠 Predictive Risk Analysis
- Simulates hazard spread (e.g., fire propagation across rooms/floors)  
- Identifies high-risk zones dynamically  
- Enables proactive decision-making before escalation  

### 🧭 Smart Evacuation System
- Generates **real-time optimized evacuation routes**  
- Avoids blocked, hazardous, or high-risk areas  
- Provides **personalized navigation guidance**  

### 🏢 3D Digital Twin Visualization
- Interactive 3D model of the building  
- Displays:
  - Fire spread simulation  
  - Room status (safe / alert / danger)  
  - Staff and guest locations  
- Improves situational awareness and control  

### 👨‍💼 Intelligent Staff Coordination
- Assigns tasks dynamically based on staff availability  
- Tracks staff positions in real time  
- Improves response efficiency and accountability  

### 📱 Dual Application System
- **Guest App**
  - Emergency alerts  
  - Evacuation guidance  
- **Staff App**
  - Task assignments  
  - Real-time updates  
  - Coordination dashboard
